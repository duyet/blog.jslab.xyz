# JSON Tools Collections

## 1 - JSON Parser/Vaidator: [http://json.jslab.xyz/tools/parser.html](http://json.jslab.xyz/tools/parser.html)
JSON Parser is a web-based tool to view, edit, and format JSON. It has various modes such as a tree editor, a code editor, and a plain text editor.

Module base on jsoneditor by [josdejong](https://github.com/josdejong/jsoneditor)
